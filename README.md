# Instant page rendering and seamless navigation for SPAs

![Instant page rendering and seamless navigation for SPAs](https://chsakell.files.wordpress.com/2018/10/ssr-spa-navigation-12.png)

##  [Read post](https://wp.me/p3mRWu-1lN)

* Describes a technique to instantly render the page on the first load without the need of an api call 
* Smooth navigation between views with entirerly different content and shared structures
* MVC actions remain clean and additional data returned only when required
